About Me

This portfolio was originally copy from a bootstrap theme, and the edited by me, there are some elements still present from the original project such as the navigation but the other .html pages and all the information within the web has been done by me.
The work of this portfolio was done in Eclipse, the code was modify there but I have used Jekyll to upload it to internet so I could see my work, my work is publised in github webs so anyone can see it, but if you want to work on your own here is a quick guide:
-First install the code and open your terminal.
-Go to your project folder (cd folder).
-If you have not install ruby, intall it (just tap ruby installer on google and download any version, 4.2 is this one).
-If you have just tap-gem install jekyll bundler
-Wait and enter-bundle install
-After this is done you just have to press-bundle exec jekyll serve --watch and watch your work on the ip they give.

After this quick guide let´s go back to the portfolio, this portfolio has 6 different html, the main page and 5 for the projects, in each file their are bottoms to download the work, the work is visual but you are free to download it if you want.
All the elements have been edited, there are pictures taken by me in this blog, but since it´s public I will rather not tell. This portfolio is the result of using .jsp, .html, and .css files, plus all the elements they include (like pictures).

As said all the work is visual but in every assesment page their are buttons to download the work individual. Enjoy my Work.
